Sistema de Compresión LZ78 + Huffman

Universidad Distrital Francisco José de Caldas - Teoría de la Información

Descripción del Proyecto

Este proyecto implementa un sistema de compresión híbrido que combina el algoritmo LZ78 (compresión basada en diccionario) con codificación Huffman (codificación óptima).

Característica Principal: El algoritmo logra tasas de compresión del 80-90% en archivos de texto con alta redundancia.

Algoritmos Implementados

LZ78

Compresión basada en diccionario adaptativo

  • Construye diccionario dinámicamente
  • Identifica patrones repetidos
  • Genera tuplas (índice, carácter)

Huffman

Codificación óptima de símbolos

  • Códigos de longitud variable
  • Símbolos frecuentes = códigos cortos
  • Optimiza representación de índices

Características del Sistema

  • Interfaz gráfica moderna con PyQt5
  • Arquitectura MVC (Modelo-Vista-Controlador)
  • Formato binario optimizado (.lz78)
  • Visualización del diccionario LZ78
  • Estadísticas detalladas en tiempo real
  • Validación robusta de archivos
  • Manejo completo de errores
Nota Técnica: El algoritmo funciona mejor con archivos que tienen alta redundancia (código fuente, logs, configuraciones). Archivos ya comprimidos (Word, Excel, ZIP) se expandirán debido a que no tienen patrones repetitivos.

Tecnologías Utilizadas

  • Python 3.10+ - Lenguaje de programación
  • PyQt5 5.15.9 - Framework de interfaz gráfica
  • Algoritmo LZ78 - Compresión basada en diccionario
  • Algoritmo Huffman - Codificación óptima
  • Formato Binario - Almacenamiento eficiente
